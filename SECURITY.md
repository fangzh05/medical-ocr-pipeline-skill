# Security Policy / 安全政策

## 中文说明

请不要上传或公开任何敏感医疗信息，包括但不限于：

- 患者隐私
- 真实病历
- 身份证号
- 手机号
- 医院号
- 检查报告原图
- 其他可识别个人身份或医疗信息的数据

如果发现安全问题，请通过 GitHub issue 联系维护者，但不要在公开内容中贴出敏感数据。请使用脱敏描述、合成样例或最小复现说明。

## English

Do not upload or publicly share sensitive medical information, including but not limited to:

- patient privacy data
- real medical records
- ID numbers
- phone numbers
- hospital IDs
- original examination report images
- any other personally identifiable or medical information

If you find a security issue, contact the maintainer through a GitHub issue, but do not post sensitive data publicly. Use de-identified descriptions, synthetic examples, or minimal reproduction notes.
