# Release Notes v0.1.0 / 发布说明 v0.1.0

## 中文

这是 `医学教材OCRskills` 的第一个公开版本。

本版本适用于医学教材、课件、题库截图和扫描 PDF 的 OCR 路由与医学符号校正。它帮助 Codex 在普通文本 OCR、公式敏感区域、表格区域、混合版面和题库结构之间选择更合适的处理路径，并在识别后修复常见医学符号扁平化问题。

### 本版本包含

- 医学 OCR 路由 skill。
- 医学符号校正规则。
- 本地医学符号校正验证脚本。
- Codex skill 结构校验说明。
- 中英双语 README、CHANGELOG、CONTRIBUTING 和 SECURITY 文档。

## English

This is the first public release of `Medical Textbook OCR Skills`.

This release is intended for OCR routing and biomedical notation correction for medical textbooks, lecture slides, question-bank screenshots, and scanned PDFs. It helps Codex choose more appropriate processing paths across plain text OCR, formula-sensitive regions, tables, mixed layouts, and question-bank structure, then correct common flattened biomedical notation after recognition.

### Included

- Medical OCR routing skill.
- Biomedical notation correction rules.
- Local biomedical notation validation script.
- Codex skill validation guidance.
- Bilingual README, CHANGELOG, CONTRIBUTING, and SECURITY documentation.
